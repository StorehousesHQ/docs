## Overview

Storehouses provides multiple ways to view and explore your portfolio items. Whether you want to see all your assets at once, focus on specific categories, or dive into detailed information about individual items, the platform offers flexible viewing options to match your needs.

## Accessing Your Items

<Steps>
  <Step title="Navigate to Dashboard">
    Log in and go to your [dashboard](https://storehouses.app/dashboard)
  </Step>

  <Step title="View All Items">
    Your dashboard displays all portfolio items by default. Click on any item card to view details.
  </Step>

  <Step title="Filter or Search">
    Use the search bar or filter options to narrow down items by category, collection, location, or other criteria
  </Step>
</Steps>

## View Modes

### Grid View

The default grid view displays items as cards with:
- Item title
- Primary photo (if uploaded)
- Current value
- Asset category
- Quick action buttons

**Best for**: Visual browsing and quick overview of your portfolio

### List View

Switch to list view for a more compact display showing:
- Item name and category
- Purchase date
- Current value
- Location
- Collection assignment

**Best for**: Scanning many items quickly and comparing values

<Tip>
Toggle between grid and list views using the view mode button in the top right corner of your dashboard.
</Tip>

## Item Detail Pages

Click any item to open its detail page, which includes:

### Basic Information

- **Item title** - Name of the asset
- **Description** - Additional notes and details
- **Category** - Asset type (precious metals, watches, etc.)
- **Quantity** - Number of identical units

### Valuation Details

<CardGroup cols={2}>
  <Card title="Purchase Information" icon="receipt">
    - Purchase price
    - Purchase date
    - Seller/source
    - Cost basis
  </Card>

  <Card title="Current Value" icon="chart-line">
    - Market value (current)
    - Insurance value
    - Value change percentage
    - Last updated timestamp
  </Card>
</CardGroup>

### Category-Specific Details

Depending on the asset category, you'll see specialized fields:

**Precious Metals**
- Metal type (gold, silver, platinum, etc.)
- Weight and unit
- Purity/fineness
- Form (coin, bar, round)
- Spot price (auto-updated)

**Luxury Watches**
- Brand and model
- Reference number
- Movement type
- Case material
- Year of manufacture

**Gemstones**
- Gem type
- Carat weight
- Cut, color, clarity
- Setting type
- Metal type

**Wine & Spirits**
- Vintage/age
- Producer/distillery
- Region
- Varietal/type
- Bottle size

**Fine Art**
- Artist name
- Medium
- Dimensions
- Year created
- Provenance

**Arms & Armor**
- Type
- Era/period
- Manufacturer
- Condition rating
- Caliber/specifications

### Location & Organization

- **Physical Location** - Where the item is stored
- **Collections** - Which collections include this item
- **Tags** - Custom labels for organization

### Photo Gallery

View all uploaded photos for an item:
- Primary photo shown largest
- Additional photos in thumbnail gallery
- Click to expand full-screen view
- Swipe or use arrows to navigate

<Warning>
Photos are stored securely and privately. They are never used for marketing or shared with third parties.
</Warning>

## Item History

### Valuation History

Track how your asset's value has changed over time:
- Historical value chart
- Date of each valuation update
- Percentage change from purchase
- Total gain/loss

**For Precious Metals**: Value updates automatically based on spot price changes

**For Other Assets**: Value updates when you manually adjust market value

### Edit History

See a complete audit trail of changes:
- Date and time of each edit
- Fields that were modified
- Previous and new values
- Who made the change (for shared portfolios)

<Tip>
Edit history helps maintain accurate records for insurance claims and tax documentation.
</Tip>

## Viewing by Category

Filter your portfolio to show only specific asset types:

1. Click the **Filter** button on your dashboard
2. Select one or more categories
3. View only items matching your selection

**Available categories**:
- Precious Metals
- Luxury Watches
- Gems
- Fine Jewelry
- Wine
- Spirits
- Fine Art
- Arms & Armor
- Other Collectibles

## Viewing by Collection

See all items in a specific collection:

1. Navigate to the **Collections** page
2. Click on any collection name
3. View all items assigned to that collection

Learn more in the [Collections guide](/features/collections).

## Viewing by Location

See all items stored at a specific location:

1. Navigate to the **Locations** page
2. Click on any location name
3. View all items currently stored there

Learn more in the [Locations guide](/features/locations).

## Search and Filters

Use the powerful search and filter features to find specific items:

### Search by Keyword

Type in the search bar to find items by:
- Title
- Description
- Brand/artist name
- Any text field

### Advanced Filters

Combine multiple filters:
- **Category** - Asset type
- **Collection** - Collection assignment
- **Location** - Storage location
- **Value Range** - Min and max value
- **Purchase Date** - Date range
- **Tags** - Custom labels

<Tip>
Save frequently used filter combinations for quick access.
</Tip>

Learn more in the [Search & Filters guide](/features/search-filters).

## Sorting Options

Sort your portfolio view by:
- **Value** (highest to lowest or vice versa)
- **Purchase Date** (newest to oldest or vice versa)
- **Title** (alphabetical A-Z or Z-A)
- **Last Updated** (most recently modified first)
- **Category** (grouped by asset type)

Click the **Sort** button and select your preferred option.

## Viewing on Mobile

The Storehouses mobile app and PWA provide:
- Responsive grid view optimized for smaller screens
- Swipe gestures for quick navigation
- Touch-optimized photo galleries
- Offline viewing of previously loaded items

<Card title="Offline Access" icon="wifi">
  Items you've viewed recently are cached for offline access. Perfect for showing your collection when you don't have internet connectivity.
</Card>

## Best Practices

<AccordionGroup>
  <Accordion title="Add Photos for Easy Recognition">
    Upload clear photos to make items instantly recognizable in grid view. Use the primary photo to show the most distinctive angle.
  </Accordion>

  <Accordion title="Use Descriptive Titles">
    Give items clear, specific titles so you can quickly identify them without opening the detail page. Include key details like brand, year, or distinguishing features.
  </Accordion>

  <Accordion title="Keep Values Current">
    Regularly review and update market values for items without automatic pricing. This ensures your portfolio total accurately reflects your holdings.
  </Accordion>

  <Accordion title="Organize with Collections">
    Group related items into collections for easier viewing. For example, create collections for "Investment Grade Coins" or "Everyday Wear Watches."
  </Accordion>

  <Accordion title="Review Item History">
    Periodically check valuation history to identify your best-performing assets and inform future acquisition decisions.
  </Accordion>

  <Accordion title="Use Tags for Flexible Organization">
    Add custom tags to items for organization beyond collections and locations. Examples: "needs-appraisal", "gift-from-dad", "insurance-pending".
  </Accordion>
</AccordionGroup>
